# Context Templates

Ten discovery blueprints, one per technology domain. Each template provides a pre-structured context for running discovery sessions tailored to that domain.

## Contents

- `ai-ml/` — Artificial Intelligence and Machine Learning
- `datalake-ingestion/` — Data Lake and data ingestion pipelines
- `migration-modernization/` — Legacy migration and modernization
- `mobile-app/` — Mobile application development
- `platform-engineering/` — Platform engineering and DevOps
- `process-automation/` — Business process automation
- `process-documentation/` — Process documentation and mapping
- `saas/` — Software as a Service products
- `system-integration/` — System integration and middleware
- `web-microservices/` — Web applications and microservices
