# cyber-security-architect

Cybersecurity and privacy architect skill. Covers personal data handling, regulatory compliance (LGPD, GDPR), PII, DPO, encryption, and authentication.

Operates in two modes: deep (when personal data is involved -- full checklist) or lean (when not applicable -- attests non-applicability). Always produces a `privacy.md` output.

This folder contains a single `SKILL.md` that defines the skill.
