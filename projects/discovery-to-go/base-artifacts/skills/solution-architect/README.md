# solution-architect

Solution architect skill for technical analysis, architecture, and financial viability. Covers technology stack definition, integration and dependency assessment, macro architecture design (monolith, microservices, serverless), 3-year TCO calculation, Build vs Buy analysis with alternatives, and technical risk evaluation.

Produces `tech-and-security.md` and `strategic-analysis.md` as outputs.

This folder contains a single `SKILL.md` that defines the skill.
