# md-validator

Validates markdown files against workspace conventions using the Python validator tool. Checks frontmatter, headings, emojis, acronyms, naming, section order, wikilinks, callouts, diagrams, and skill fields. Accepts a single file or an entire directory for recursive validation.

Supports a `--fix` mode for automatic correction of common issues.

This folder contains a single `SKILL.md` that defines the skill.
