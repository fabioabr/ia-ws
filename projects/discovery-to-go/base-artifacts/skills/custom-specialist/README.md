# custom-specialist

Dynamic meta-skill that assumes expertise in any technical domain on demand. Called by other agents or users when deep domain knowledge is needed -- for example, Kafka streaming, cloud architecture, ML/MLOps, ERPs, or sector-specific compliance (SOX, PCI-DSS, HIPAA).

Assumes the specialist role, validates information plausibility in the domain, and marks implausible claims as contested.

This folder contains a single `SKILL.md` that defines the skill.
