# Skills

Global skills for AI assistants. Each subfolder contains a single `SKILL.md` defining the skill's behavior, inputs, outputs, and constraints.

## Available Skills

| Skill | Description |
|-------|-------------|
| `10th-man` | Devil's advocate that challenges assumptions and validates decisions through divergent review |
| `chart-specialist` | Data visualization consultant that recommends the best chart type for a given dataset |
| `custom-specialist` | On-demand domain expertise for any technical area (Kafka, SAP, ML, compliance, etc.) |
| `cyber-security-architect` | Privacy and security analysis covering LGPD, GDPR, PII, encryption, and compliance |
| `html-writer` | Converts markdown documents into self-contained HTML reports with dark/light themes |
| `md-validator` | Validates markdown files against workspace conventions (frontmatter, headings, emojis, etc.) |
| `md-writer` | Formats and polishes markdown files following workspace conventions |
| `po` | Product analysis covering vision, personas, value proposition, and organization |
| `solution-architect` | Technical architecture, stack selection, TCO calculation, and Build vs Buy analysis |
