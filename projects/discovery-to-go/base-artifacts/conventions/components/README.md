# Components

HTML component standards for self-contained report generation, covering layout, data display, and interactive elements.

## Files

- `alerts.md` — Alert and notification component styles
- `badge.md` — Badge component for labels and status indicators
- `card.md` — Card container component
- `header-footer.md` — Page header and footer layout
- `stat-card.md` — Statistical highlight card component
- `table.md` — Table styling and structure rules
- `tabs.md` — Tabbed content component
