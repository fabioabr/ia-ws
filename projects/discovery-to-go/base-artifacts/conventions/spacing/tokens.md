# Spacing Tokens

Spacing scale based on a 4px base unit.

## Standard

| Token | Value |
|-------|-------|
| xs | 4px |
| sm | 8px |
| md | 16px |
| lg | 24px |
| xl | 32px |
| 2xl | 48px |

Base unit: **4px**
