# Context Templates

Copy of the 10 discovery blueprints used as template reference during report generation.

Each subfolder contains a `discovery-blueprint.md` tailored to a specific solution domain.

## Blueprints

- **ai-ml** — AI and Machine Learning projects
- **datalake-ingestion** — Data lake and ingestion pipelines
- **migration-modernization** — Legacy migration and modernization
- **mobile-app** — Mobile application development
- **platform-engineering** — Platform engineering and developer platforms
- **process-automation** — RPA and process automation
- **process-documentation** — Process documentation and governance
- **saas** — SaaS product development
- **system-integration** — System integration projects
- **web-microservices** — Web applications with microservices architecture
