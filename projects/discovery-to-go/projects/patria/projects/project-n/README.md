# Run Sample -- FinTrack Pro

Complete end-to-end example of a Discovery-to-Go pipeline run for **FinTrack Pro**, a SaaS platform for automated multi-bank financial consolidation targeting SMBs.

This sample run uses the `saas` context-template and completes in a single iteration covering all three phases: Discovery, Challenge, and Delivery.

## Contents

| Item | Description |
|------|-------------|
| `pipeline-state.md` | Pipeline state tracker, updated by the orchestrator after each phase |
| `setup/` | Run configuration (briefing, config, customization overrides) |
| `iterations/` | Iteration history with logs and phase results |
| `delivery/` | Final delivery outputs (report, plan, HTML) |
