# Starter Kit

Artefatos para iniciar um novo projeto de discovery. Copie os templates necessários e preencha.

## Conteúdo

| Arquivo | Descrição | Quando usar |
|---------|-----------|-------------|
| `briefing-template.md` | Modelo de briefing inicial — preencha com as informações do projeto | Obrigatório — sem briefing o pipeline não inicia |

## Como usar

1. Copie o `briefing-template.md` para a pasta do seu projeto
2. Renomeie para `briefing.md`
3. Preencha todas as seções obrigatórias (1 a 6)
4. Invoque o orchestrator: `/orchestrator briefing.md`

## Diferença do template em dtg-artifacts

O template em `dtg-artifacts/templates/draft-templates/briefing-template.md` é a versão técnica consumida pelo pipeline. Este aqui no starter-kit é a versão amigável para o cliente preencher — com exemplos, tabelas prontas e linguagem acessível.
