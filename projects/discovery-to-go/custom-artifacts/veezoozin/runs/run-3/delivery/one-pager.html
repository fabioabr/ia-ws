Premissas
REG-EXEC-07
Premissas Estruturais do Projeto
7 premissas
BYOK obrigatório a partir do Pro — custo de LLM 100% do tenant. mAInd Tech não paga tokens de inferência.
Pay-per-use integral — infraestrutura 100% serverless (GCP). Custo variável ~R$ 55-70/tenant/mês.
1 desenvolvedor sênior + Claude Code — output equivalente a ~3 devs full-stack. Sem contratações até R$ 25K MRR.
BigQuery como único banco analítico no MVP — multi-banco (PostgreSQL, MySQL) somente na Fase 2.
Investimento de ~R$ 184K até break-even — mês 19 (cenário base com 1a contratação no mês 13).
Precisão NL-to-SQL meta >85% — kill criterion: <70% leva a pivot. 10th-man recomenda meta escalonada (75% → 80% → 85%).
Bootstrapping — sem investimento externo. Receita orgânica via modelo freemium + conversão Free → Pro.
