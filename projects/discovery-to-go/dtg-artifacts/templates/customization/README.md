# Customization

Customizable defaults for the Discovery Pipeline. Projects can copy any of these files into their local `customization/` folder to override the global behavior.

| File | What it controls |
|------|-----------------|
| `scoring-thresholds.md` | Minimum averages and per-dimension floors for auditor and 10th-man gates; includes profiles for standard, POC, and high-risk projects |
| `iteration-policy.md` | Max iterations, stagnation threshold, default HR loop behavior, and abort rules |
| `final-report-template.md` | Structure of the consolidated delivery report — mandatory sections, one-pager layout, and tone per section |
| `html-layout.md` | Regions, order, and layout mode (full-width, grid) for the HTML delivery report |
| `human-review-template.md` | Text, format, and fields for the Human Review section appended at the end of each round |
