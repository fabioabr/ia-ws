# Draft Templates

Templates de artefatos usados pelo orchestrator ao criar o scaffold da run e pelos agentes ao gerar outputs.

## Conteúdo

| Template | Usado por | Descrição |
|----------|-----------|-----------|
| `briefing-template.md` | Humano (pré-pipeline) | Modelo para o briefing inicial do cliente |
| `audit-report-template.md` | Auditor (Fase 2) | Estrutura do relatório de validação convergente (5 dimensões) |
| `challenge-report-template.md` | 10th-man (Fase 2) | Estrutura do relatório de validação divergente |
| `change-request-template.md` | Orchestrator (HR loop) | Formato de change request quando humano reprova |
| `iteration-setup-template.md` | Orchestrator (cada iteração) | Configuração de cada nova iteração |
