# Customization

Customization overrides for this pipeline run. These files tailor the pipeline behavior, context, report structure, and scoring rules to the specific project needs.

## Contents

| Directory | Description |
|-----------|-------------|
| `context-themes/` | Context theme definitions (currently empty -- using defaults) |
| `current-context/` | Active context-template applied to this run (saas) |
| `report-templates/` | Report structure and human-review templates |
| `rules/` | Run-specific rules for scoring and iteration policy |
