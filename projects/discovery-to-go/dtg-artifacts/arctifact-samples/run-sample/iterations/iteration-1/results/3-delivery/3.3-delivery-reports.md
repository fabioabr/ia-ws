## Design System Aplicado

O relatorio HTML foi gerado com o design system padrao do pipeline, incluindo:

- **Tipografia** — Inter (corpo) + JetBrains Mono (codigo).
- **Paleta** — Tema corporativo do workspace, variantes claro/escuro.
- **Componentes** — Tabelas estilizadas, callouts, badges de status, barras de progresso.

---

## Funcionalidades

- **Tema claro/escuro** — Toggle manual + deteccao automatica via `prefers-color-scheme`.
- **Diagramas Mermaid** — Renderizacao client-side de arquitetura e fluxos.
- **Navegacao lateral** — Table of contents fixa com scroll-spy.
- **Self-contained** — CSS, fontes e scripts embutidos. Visualizacao offline sem dependencias externas.
- **Responsivo + impressao** — Layout adaptavel (375px-1920px) com stylesheet `@media print`.
