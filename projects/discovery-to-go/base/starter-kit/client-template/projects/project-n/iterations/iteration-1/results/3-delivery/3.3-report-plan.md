## Decisoes de Layout

O report-planner analisou o `delivery-report.md` consolidado e o `html-layout.md` para determinar quais regions ativar e em que ordem:

- **Sections ativas:** 10 secoes padrao + 2 regions domain-specific (SaaS).
- **Regions domain-specific incluidas:** REG-DOM-SAAS-01 (Pricing table), REG-DOM-SAAS-02 (Tenancy strategy) — ativadas pelo context-template `saas`.
- **Regions opcionais omitidas:** Nenhuma region opcional foi solicitada pelo cliente.
