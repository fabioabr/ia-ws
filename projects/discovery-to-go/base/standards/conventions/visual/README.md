# Visual

Design tokens, componentes HTML e configurações de gráficos. Define como as coisas se apresentam visualmente.

## Contents

- `colors/` — Paleta de cores, chart palette, sequential palette, contraste
- `typography/` — Escala tipográfica e fontes
- `spacing/` — Tokens de espaçamento, border-radius, shadows
- `icons/` — Biblioteca de ícones (Remix Icon)
- `responsive/` — Breakpoints e ajustes responsivos
- `components/` — Componentes reutilizáveis (card, table, alerts, badge, tabs, stat-card, header-footer)
- `charts/` — Configuração Chart.js
