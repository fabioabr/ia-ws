## Tokens

| Token | Value | Componentes |
|-------|-------|-------------|

| sm | `6px` | Inputs, theme toggle, offcanvas menu items |

| lg | `10px` | Cards, stat cards, stat card icons, section icons, phase boxes, code blocks, wireframe |

| full | `20px` | Badge version, section count |
