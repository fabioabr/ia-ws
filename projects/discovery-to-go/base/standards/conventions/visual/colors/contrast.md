# Color Contrast Rules

Regras de contraste para texto sobre fundos coloridos.

## Background Utility Classes

| Class | Background | Text (dark) | Text (light) |
|-------|-----------|-------------|--------------|
| `.bg-primary` | `var(--primary)` | `#1A1923` | `#1A1923` |
| `.bg-success` | `var(--success)` | `#1A1923` | `#1A1923` |
| `.bg-warning` | `var(--warning)` | `#1A1923` | `#1A1923` |
| `.bg-teal` | `var(--teal)` | `#1A1923` | `#1A1923` |
| `.bg-orange` | `var(--orange)` | `#1A1923` | `#1A1923` |
| `.bg-secondary` | `var(--secondary)` | `#1A1923` | `#1A1923` |
| `.bg-info` | `var(--info)` | `#FFFFFF` | `#1A1923` |
| `.bg-danger` | `var(--danger)` | `#FFFFFF` | `#1A1923` |
| `.bg-purple` | `var(--purple)` | `#FFFFFF` | `#1A1923` |

## Card Body Text

| Theme | Color | Opacity |
|-------|-------|---------|
| Dark | `var(--text)` — `#E0DFE3` | `0.85` |
| Light | `var(--text)` — `#1A1923` | `0.85` |
