# Sequential Color Palette

Sequência cíclica para elementos ordenados. Volta à posição 1 após o último item.

## Standard

| Position | Color | CSS Class |
|----------|-------|-----------|
| 1 | Primary (`#4DA8DA`) | `.bg-primary` |
| 2 | Success (`#0ED145`) | `.bg-success` |
| 3 | Warning (`#F4AC00`) | `.bg-warning` |
| 4 | Danger (`#8B1A1A`) | `.bg-danger` |
| 5 | Purple (`#9B96FF`) | `.bg-purple` |
| 6 | Orange (`#FF9473`) | `.bg-orange` |
| 7 | Teal (`#2DD4BF`) | `.bg-teal` |
| 8 | Info (`#2EB5F5`) | `.bg-info` |

## Aplica-se a

- Phase flows
- Section icons
- Stat card icons
- Timeline dots (`.dot-primary`, `.dot-success`, etc.)
- Bar segments
- KPIs e badges
