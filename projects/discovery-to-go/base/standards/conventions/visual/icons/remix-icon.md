# Remix Icon

## Library

- **Package:** Remix Icon `remixicon@4.1.0`
- **CDN:** `https://cdn.jsdelivr.net/npm/remixicon@4.1.0/fonts/remixicon.css`

## Sizes

| Context | Size |
|---------|------|
| Card header icons | `1rem` |
| Tabs / Section headers | `1.1rem` |
| Alerts | `1.1rem` |
| Stat cards | `1.3rem` |
| Back-to-top / Floating menu | `1.3rem` |
| Offcanvas menu close | `1.2rem` |
| Offcanvas menu items | `1.1rem` (width `20px`, centered) |
| Footer seal | `1rem` |
| Phase flow arrows | `1.4rem` |

## Color

Herdada do elemento pai (`bg-*` class) ou do token semântico do contexto.

## Icon Width (menu items)

| Context | Width | Alinhamento |
|---------|-------|-------------|
| Tab links | — | `margin-right: 6px` |
| Offcanvas menu | `20px` | `text-align: center` |
| Card header | — | Dentro de icon box `32x32px` |
