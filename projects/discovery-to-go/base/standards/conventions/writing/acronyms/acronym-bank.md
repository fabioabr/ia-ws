# Acronym Bank

Banco centralizado de siglas. Ao gerar qualquer documento (HTML ou MD), consulte esta lista para obter o significado correto. Se encontrar uma sigla nova, adicione-a aqui.

> [!info] Formato
> - **Sigla**: texto exibido ao leitor
> - **title**: texto do tooltip (`<abbr title="...">`)
> - **Significado**: forma extensa
> - **Categoria**: T = Técnica, N = Negócio, R = Regulatória, O = Organizacional

---
