# Report Planner

Visual report planning skill for Phase 3 (Delivery) of the Discovery Pipeline.

Defines how the delivery report will be rendered visually: which regions appear, in what order, with what layout (full-width, grid), and which visualization type to use for each region (card, table, chart). Runs after the consolidator generates `delivery-report.md` and before the html-writer produces the final HTML.

Generates `report-plan.md` with the complete visual specification.

This folder contains `SKILL.md` with the full skill specification.
