# Skills

Pipeline-specific skills for Discovery-to-Go. Each subfolder contains a `SKILL.md` with full specification.

| Skill | Phase | Role |
|-------|-------|------|
| `customer` | Phase 1 — Discovery | Client simulator that answers specialist questions based on the briefing |
| `auditor` | Phase 2 — Challenge | Convergent quality gate — audits drafts against 5 weighted dimensions |
| `orchestrator` | All phases | Central coordinator — manages state, transitions, iterations, and HR loop |
| `pipeline-md-writer` | Phase 3 — Delivery | Polishes approved drafts into formatted Markdown documents |
| `report-planner` | Phase 3 — Delivery | Plans visual layout and regions for the HTML delivery report |
| `consolidator` | Phase 3 — Delivery | Merges all results into `delivery-report.md` with executive overview |

Additionally, `orchestrator-workspace/` holds evaluation data for the orchestrator skill.
