# Pipeline MD Writer

Markdown formatting skill for Phase 3 (Delivery) of the Discovery Pipeline.

Transforms approved results (blocks 1.1--1.8, 2.1--2.2) into polished Markdown documents following workspace conventions. Runs first in Phase 3, before the consolidator. This is a pipeline-specific extension of the global md-writer skill and inherits all its formatting rules.

Generates `3.1-markdown-documents.md`.

This folder contains `SKILL.md` with the full skill specification.
