# Index Structure

Standard format for index.md files.

## Standard

Every area must have an `index.md` file.

### Format

1. **H1 title** — area name
2. **Brief description** — one or two sentences
3. **Grouped wikilinks** — organized by topic, each with a short description

### Ordering

- Links are in **logical order** (not alphabetical)
- Group related items under subheadings
