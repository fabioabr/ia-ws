# Document Sections

Standard sections and lifecycle for documents.

## Standard

### Status Lifecycle

| Status | Emoji | Description |
|---|---|---|
| rascunho | :pencil2: | Draft, in progress |
| ativo | :white_check_mark: | Active, current version |
| arquivado | :file_cabinet: | Archived, no longer maintained |
| obsoleto | :no_entry: | Obsolete, superseded |

### Deviation Log Section

Table with the following columns:

| Column | Description |
|---|---|
| Timestamp | When the deviation occurred |
| Regra | Rule that was deviated from |
| Sugestão da IA | AI's original suggestion |
| Decisão do Usuário | User's actual decision |
| Motivo | Reason for the deviation |
