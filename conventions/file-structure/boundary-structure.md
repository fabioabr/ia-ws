# Boundary Structure

Standard file structure for project boundaries documentation.

## Standard

### Location

```
project/docs/2-boundaries/
```

### Required Files

| File | Purpose |
|---|---|
| `index.md` | Boundary overview and navigation |
| `technology.md` | Technology constraints and choices |
| `security.md` | Security boundaries and policies |
| `process.md` | Process boundaries and workflows |
| `business.md` | Business rules and constraints |
| `architecture.md` | Architecture decisions and limits |

### Format

Each file follows markdown-writing rules with frontmatter.
