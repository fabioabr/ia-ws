# Remix Icon

Icon library standard for all UI components.

## Standard

- **Library:** Remix Icon (`remixicon@4.1.0`)

### Sizes

| Context | Size |
|---------|------|
| Card headers | 1rem |
| Tabs / Section headers | 1.1rem |
| Stat cards | 1.3rem |

### Color

Inherited from the `bg-*` parent element.
