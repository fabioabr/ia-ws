# Box Shadows

Elevation shadow tokens per theme.

## Standard

### Dark Theme

Dark theme uses a blue glow effect.

| Level | Value |
|-------|-------|
| L1 | `0 2px 12px rgba(46,181,245,0.06), 0 1px 4px rgba(0,0,0,0.4)` |
| L2 | `0 4px 16px rgba(46,181,245,0.1), 0 2px 8px rgba(0,0,0,0.5)` |
