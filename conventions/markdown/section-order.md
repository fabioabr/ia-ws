# Section Order

Standard ordering of sections in a Markdown document.

## Standard

1. **Frontmatter** (YAML block)
2. **Content** (H1 title + body sections)
3. **Documentos Relacionados** (wikilinks to related docs)
4. **Desvios de Behavior** (any deviations from standard rules, with justification)
5. **Historico de Alteracoes** (changelog — always last)

The changelog is **always** the final section in the document.
