# Callouts

Obsidian callout types and syntax.

## Standard

Use only these callout types:

| Type | Purpose |
|---------|-------------------------------|
| note | General information |
| tip | Best practices, suggestions |
| info | Contextual details |
| warning | Caution, potential issues |
| danger | Critical risk, breaking change |
| example | Concrete usage demonstration |

## Syntax

```markdown
> [!note]
> General information here.

> [!tip]
> A helpful suggestion.

> [!info]
> Additional context or detail.

> [!warning]
> Proceed with caution.

> [!danger]
> Critical risk — may cause data loss.

> [!example]
> Step-by-step demonstration here.
```
