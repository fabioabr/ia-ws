# Semantic Emojis

Emojis com função semântica para headings e destaques em documentos .md e relatórios HTML.

## Standard

| Emoji | Significado | Uso |
|-------|-------------|-----|
| 🎯 | Objetivo / Meta | Seções de goals, OKRs |
| ⚠️ | Atenção / Warning | Riscos, avisos |
| ❌ | Proibido / Crítico | Regras negativas, bloqueios |
| ✅ | Aprovado / OK | Validações, confirmações |
| 📋 | Lista / Checklist | Seções de itens, tarefas |
| 🔗 | Documentos Relacionados | Seção de backlinks |
| 📜 | Histórico de Alterações | Changelog (sempre última seção) |
| 🏗️ | Arquitetura | Seções técnicas de arquitetura |
| 🔒 | Segurança | Privacidade, compliance, LGPD |
| 📊 | Métricas / Dados | Dashboards, KPIs, analytics |
| 📅 | Cronograma / Prazo | Timelines, deadlines |
| 💰 | Custo / TCO | Seções financeiras |
| 🏛️ | Princípio / Fundamento | Regras fundamentais |
| 🔍 | Revisão / Auditoria | Seções de validação |
| 📄 | Documento | Referência a arquivo |
| 🔮 | Futuro / Roadmap | Categorias futuras, planejamento |
| 📚 | Índice / Catálogo | Seções de navegação |
| 🗂️ | Estrutura / Organização | Pastas, taxonomia |

## Regra de uso

- Usar apenas em **headings H2** (seções principais)
- Nunca decorativo — sempre com função semântica
- Máximo 1 emoji por heading
- Não usar no H1 (título do documento)
