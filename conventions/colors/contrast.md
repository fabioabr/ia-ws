# Color Contrast Rules

Text color pairing rules for accessible contrast.

## Standard

| Background Type | Text Color |
|----------------|------------|
| Light backgrounds (blue, green, orange, teal, warning) | #1A1923 |
| Dark backgrounds (danger, purple, info) | #FFFFFF |

### Card Text Opacity

| Theme | Text Color | Opacity |
|-------|------------|---------|
| Dark | #E0DFE3 | 0.85 |
| Light | #1A1923 | 0.85 |
