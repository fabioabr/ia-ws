# Tag Taxonomy

Standard for creating and applying tags to documents.

## Standard

- **Language:** pt-BR without accents
- **Format:** kebab-case, singular
- **Quantity:** 2-5 tags per document

### Category Tags

`core` | `processo` | `documentacao` | `projeto` | `ferramenta`

### Theme Tags

`markdown` | `taxonomia` | `organizacao` | `versionamento` | `obsidian`

### Absorbed English Words (exceptions)

These English terms are used as-is, without translation:

`markdown` | `git` | `scrum` | `kanban` | `sprint` | `deploy` | `feedback` | `dashboard`

## Examples

| Document | Tags |
|------------------------------|--------------------------------------|
| Versioning guide | `documentacao`, `versionamento` |
| Git workflow | `processo`, `git` |
| Kanban board setup | `ferramenta`, `kanban`, `organizacao`|
