# Typography Scale

Type scale tokens for all text elements.

## Standard

| Token | Size | Weight | Line Height |
|-------|------|--------|-------------|
| Display | 1.5rem | 700 | 1.2 |
| H1 | 1.15rem | 600 | 1.3 |
| H2 | 0.95rem | 600 | 1.3 |
| Body | 0.88rem | 400 | 1.5 |
| Body Small | 0.82rem | 400 | 1.5 |
| Caption | 0.72rem | 600 | 1.4 |
| Code | 0.78rem | 400 | 1.6 |

### Font Families

| Usage | Font |
|-------|------|
| Interface | Poppins |
| Code | Consolas |
